# 人造語言生成器！

這是一個專為造語者設計的整合開發環境，讓你擺脫複雜的指令，專注於構建你的語言世界。

---

## 🛠️ 快速開始 (只需 3 步)

1. **安裝 Python**：確保你的電腦已安裝 Python 3.x。
2. **雙擊啟動**：在專案資料夾中找到 `conlang.bat` 並雙擊。
3. **開始使用**：系統會自動安裝必要插件（Flask / PyYAML）並開啟瀏覽器介面。

> **提示**：啟動後的黑色視窗是後端服務，使用期間請勿關閉，縮小即可。

---

# 🌍 Conlang 語言建構工具：技術手冊 (v1.0)

這是一個 IPA 導向的語言建構工具，旨在提供從音標選取、頻率管理、單字生成到構詞規則維護的一站式解決方案。

## 🏠 首頁：專案管理
* **專案建立**：開始前必須先建立一個專案。
* **更名注意**：目前介面暫不支援專案更名，但可以直接手動更改專案資料夾名稱，系統會自動同步。

## ⚙️ 語法配置 (Grammar)
1. **設定**：挑選語言所需的語法特徵。
   * **重點**：若名詞或動詞需要「性、數」等變化，必須先在「通用特性」欄位點選，後續分頁才能進行關聯選擇。
   * *註：部分過濾器暫時移除，後續版本將補回。*
2. **排序**：可針對特定項目設定字典式的預設排序，這將影響「構詞管理」中的表格顯示順序。

## 🎙️ 音標系統 (IPA Picker)
* **核心邏輯**：目前版本為 **IPA 導向**，僅支援標準 IPA 符號選取。
* **限制**：暫不支援歷史字母或自訂正字法（預計於後續更新加入）。

---

## 📊 音標管理 (IPA Management)

### **1. 總覽 (Overview)**
* **字母表順序**：可自定義排序（目前僅影響顯示順序）。
* **出現頻率**：調整各個 IPA 在所屬類別（子音/母音）中的生成頻率。
* **機率算法**：若未指定音節類型，單個音素出現機率為：
  $$P = \frac{\text{IPA.frequency}}{\sum \text{Category.frequency}}$$

### **2. IPA 分類區 (Categories)**
#### 📖 使用說明
* **自動生成**：在「音標系統」儲存後，系統會自動根據發音特徵產生預設分類（如 P, N, F 等）。
* **手動新增**：點擊「新增類別」可建立自定義代碼。
* **編輯模式**：點擊分類塊（**藍色邊框**）即進入編輯狀態。此時點擊下方的 **IPA 符號池** 即可新增或移除該分類中的符號。

#### 📝 欄位定義與規範
* **CODE (代碼)**：用於規則寫作的簡寫（如 `P -> B / V_V`）。建議使用**大寫英文字母**。
* **COMMENT (註解)**：說明代碼意義（如 `Plosives` 或 `脣音`），不參與程式運算。
* **捲動區域**：分類列表固定高度為 **300px**，可直接於內部捲動。
* **特殊限制**：
  * **C、V** 預設為子音與母音，**無法更改**。
  * **代碼唯一性**：CODE 不可重複，否則會彈出警告。
  * **儲存流程**：目前不支援在建立分類時同步調整註解。請**先更新分類並儲存設定，再回來調整註解內容**。

#### 📋 預設分類表
| 代碼 (CODE) | 註解 (COMMENT) | 包含範圍 (發音部位/方法) |
| :--- | :--- | :--- |
| **P** | Plosives | 塞音 (如 p, t, k, b, d, g) |
| **N** | Nasals | 鼻音 (如 m, n, ŋ) |
| **F** | Fricatives | 擦音 (如 f, s, x, v, z) |
| **L** | Liquids/Approximants | 流音、近音、顫音 (如 l, r, j, w) |
| **B** | Labials | 雙唇音、唇齒音 (唇部發音) |
| **K** | Coronals | 齒音、齒齦音、齒齦後音 (舌尖/舌葉發音) |
| **G** | Gutturals/Back | 軟顎音、小舌音、喉音 (後部發音) |
| **S** | Sibilants | 噝音 (具有高頻特徵的擦音) |
| **O** | Others | 其他未歸類之音素 |

## ✍️ 單字生成 (Generator)
系統參照音標管理的權重邏輯，依據音節格式輸入（如 `CVC`）與音節數範圍隨機生成。
1. **批量生成**：生成大量無意義字串，使用者再自行篩選加入字典。
2. **一對一新建**：對應翻譯清單直接生成。目前預設詞性皆為「名詞」，需手動修改。

---

## 📖 字典 (Dictionary)
* 提供直覺的單字編輯、刪除與新增功能。

---

## 🛠️ 構詞管理 (Morphology)
* **動詞**：邏輯寫作中，功能尚未開放。
* **名詞**：可根據「語法配置」中選取的特徵（維度）建立表格。在每個空格中填入對應的變化規則即可。
